ViceVersa is a two-sided care coordination platform connecting patients and healthcare providers, built for the **Seattle Microsoft Divergent Hackathon 2026**.

Nikhil developed the provider-facing application (`care-sync-provider-viceversa`), handling scheduling, patient communication, and care workflow management. The app uses React Native and AI-driven features to streamline provider workloads in underserved healthcare settings.

The project placed **3rd in the Healthcare track** at the Microsoft Divergent Hackathon.
