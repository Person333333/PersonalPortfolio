# Nikhil V — Project Portfolio

Bold, editorial-style portfolio site built with Astro + Decap CMS, hosted on Netlify.

## Local dev

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`

## CMS

Decap CMS lives at `/admin`.
On Netlify, enable Identity + Git Gateway to allow content editing.

## Deployment

Netlify builds with `npm run build` and outputs to `dist/`.
